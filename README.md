## 项目简介
代码生成器（测试版）是一款测试版的 Springboot 项目 CRUD 方法生成工具，目前依旧存在部分 bug，但已经可以进行初步的使用。

## 项目特点
- 代码遵循阿里巴巴开发规范。

## 技术介绍
SpringBoot + MyBatis + Mysql
